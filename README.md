# Sky Shards (Небесные осколки)

Многопользовательская браузерная 3D‑игра в жанре survival‑roguelike. Сервер — Node.js + Socket.IO + Cannon.js (авторитарная модель). Клиент — Three.js (WebGL рендер с веткой WebGPU), Vite‑сборка.

> «Острова парят над Бездной. Гравитационный якорь иссякает. Введите имя и попробуйте выжить.»

## Структура

```
/server   — Node + Express + Socket.IO + Cannon.js
/client   — Vite + Three.js (FPS-камера, инстансинг, LOD, постобработка)
/shared   — общие типы и константы (PlayerState, Snapshot, Recipe, ...)
```

## Запуск

```bash
npm install
npm run dev
```

Скрипт запускает сервер на `http://localhost:3001` и Vite-клиент на `http://localhost:5173` параллельно (через `concurrently`). Откройте клиент в браузере, введите имя и нажмите «Подключиться».

## Бюджеты производительности

- **FPS:** 60 на GeForce GTX 1050 / Intel Iris Xe в 1080p.
- **Треугольники:** ≤ 400 000 на кадр.
- **Draw calls:** ≤ 250.
- **VRAM:** ≤ 500 МБ.
- **Частицы:** ≤ 3 000 одновременно (только GPU).

Все цифры выводятся в HUD: `FPS / Draws / Tri`.

## Архитектура

- **Серверная авторитарность.** Сервер тикает 20 Гц, клиент шлёт ввод 20 Гц, получает снапшоты 20 Гц.
- **Интерполяция.** Клиент рисует мир с задержкой 100 мс, между двумя последними снапшотами.
- **Client-side prediction.** Локальный игрок двигается мгновенно, при расхождении с сервером плавно подтягивается.
- **Дельта-снапшоты.** Острова шлются раз в 30 тиков, остальное — каждый тик.
- **Roguelike.** Смерть перманентна. Сессия заканчивается падением островов или гибелью всех игроков.

## Игровые системы (по спринтам)

1. **Спринт 1 — Технологический.** Сервер + клиент, эталонный остров, WASD-капсула.
2. **Спринт 2 — Мир.** Процедурная генерация островов (3D simplex), биомы, инстансинг растительности, инвентарь, добыча, базовый крафт (топор, кирка).
3. **Спринт 3 — Якорь и энергия.** Гравитационный якорь, лоток кристаллов, расширение радиуса, аварийный стабилизатор. Соло-автоматизация (авто-подача из лотка). Планер, крюк-кошка, канатные дороги.
4. **Спринт 4 — Враги.** Скаты, гарпии, пожиратели энергии, големы. Ночные волны, цикл дня/ночи, голод, смерть.
5. **Спринт 5 — Шторм и боссы.** Штормы, буревестник, Небесный кит, Древень бурь. LOD, object pooling, балансировка.

## Развитие

- Мобильное управление (gamepad/touch)
- Battle Pass / косметика (поле `purchasedItems[]` уже зарезервировано)
